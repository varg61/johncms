Алгоритм определения языка
---------------------------
Изначально, язык задается в системных настройках (Админ панель).

"Выбирать автоматически" разрешено.
1) При первом входе, язык выбирается автоматически, по сигнатурам браузера и из списка доступных языков.
   Если запрашиваемого языка нет в системе, то будет установлен тот, что указан в системных настройках.
   ISO код языка записывается в кэш сессии для дальнейшего использования.
2) Если юзер авторизован на сайте и у него в личных настройках указан язык, будет использован он.
   ISO код языка записывается в кэш сессии для дальнейшего использования.
3) Если посетитель (независимо от того, авторизован, или нет) сменит язык с помощью переключателя на Главной,
   то на все время жизни сессии, будет использован он, независимо от личных настроек.
   ISO код языка записывается в кэш сессии для дальнейшего использования.

"Выбирать автоматически" запрещено.
Для всех посетителей будет установлен язык, указанный в системных настройках.
Возможность переключения языков будет недоступна.
