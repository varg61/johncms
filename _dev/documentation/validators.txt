Вызов:
$check = new Validate($type, $value, $option);

$type - тип валидатора, из списка имеющихся.
$value - проверяемое значение.
$option - необязательный массив с дополнительными параметрами.

Возвращаемые значения:
$check->is
TRUE - если валидация прошла
FALSE - если валидация не прошла

Сообщения об ошибках:
$check->error[] - массив с сообщениями об ошибках валидации (если валидация не прошла)

================================================================================
lenght
--------------------------------------------------------------------------------
Валидация длины строки на вхождение в заданный диапазон.

Параметры (возможно указывать один из них):
min - Минимально допустимая длина строки
max - Максимально допустимая длина строки

ПРИМЕР: проверяет длину строки на минимум 10 и максимум 20 символов
$value = 'Test string';
$option['min'] = 10;
$option['max'] = 20;
$check = new Validate('lenght', $value, $option);
if($check->is){
  // Проверка ппрошла
} else {
  // Проверка не прошла, выводим сообщения об ошибках
  foreach($check->error as $error){
    echo 'ERROR:' . $error . '<br/>';
  }
}

================================================================================
numeric
--------------------------------------------------------------------------------
Проверка, является ли строка числом и находится ли это число в заданном диапазоне.

Параметры (не обязательные):
min - Минимально допустимое число
max - Максимально допустимое число

ПРИМЕР: проверяет, является ли $value числом не менее 100 и не более 500
$value = 200;
$option['min'] = 100;
$option['max'] = 500;
$check = new Validate('numeric', $value, $option);
if($check->is){
  // Проверка ппрошла
} else {
  // Проверка не прошла, выводим сообщения об ошибках
  foreach($check->error as $error){
    echo 'ERROR:' . $error . '<br/>';
  }
}
