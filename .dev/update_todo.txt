В редакторе языков поменять путь и название создаваемого файла





------------------------------------------------------------
Предварительные проверки (отдельный файл check.php)
------------------------------------------------------------
1) Проверяем, есть ли совпадающие записи в поле `name` и `name_lat`
   Если есть, выдаем список со ссылками на профили.
2) Проверяем, есть ли совпадающие записи в поле E-mail.
   Если есть, то вместо них, вписываем пустые значения.

------------------------------------------------------------
Процедура обновления
------------------------------------------------------------
1) Выполняем SQL файл
2) Конвертируем пользователей, переносим данные из таблицы `users` в таблицу `users`:
   - Если в старой таблице поле `name` равно транслитерированному в маленькие буквы `name_lat` то в новую таблицу записываем `name`, если нет, то `name_lat`
   - Если поле `mail` не пустое и там корректный E-mail адрес, то записываем его в новую таблицу
   - Конвертируем пол
   - Если корректно указан день рожденья, то конвертируем его в формат даты и записываем в новую таблицу `cms_user_fields`
   - Остальные данные переносятся как есть
3) DROP TABLE `users`;


------------------------------------------------------------
Очистка устаревших данных
------------------------------------------------------------
Из старой галереи удалить каменты из таблицы (WHERE `type` = 'km')
