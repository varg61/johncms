ВНИМАНИЕ!!!
Данный пакет предназначен для обновления JohnCMS с версии 4.0.1 до 4.1.0
С другими версиями несовместим!

Если Вы использовали на своем сайте какие-либо моды, затрагивающие файлы
которые входят в данный пакет, то возможность обновления и саму процедуру
согласовывайте с авторами модов.

Перед обновлением, ОБЯЗАТЕЛЬНО сделайте резервные копии сайта и базы данных.

ОБНОВЛЕНИE
1) Полностью удаляем следующие папки:
   /panel
   /sitemap

2) Загружаем распакованный пакет обновлений на сервер, с заменой всех файлов

3) Запускаем процедуру обновления по адресу http://ваш_сайт/install
   Устанавливаем обновление для всех языков Вашего сайта
   
4) Удаляем папку /install

--------------------------------------------------------------------------------

ATTENTION!!!
This package is designed to update JohnCMS with version 4.0.1 to 4.1.0
Not compatible with other versions!

If you used on your site any mods that affect files within this package,
the ability to update and the process itself coordinate with the authors of the mods.

Prior to upgrading, BE SURE to backup your site and database.

OBNOVLENIE
1) Completely remove the following folders:
   /panel
   /sitemap

2) Upoad the unzipped Service Pack to the server, and replace all files

3) Run the upgrade procedure at http://your_site/install
   Install update for all languages on your site
   
4) Delete folder/install
